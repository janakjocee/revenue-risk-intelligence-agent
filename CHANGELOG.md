# Changelog

## v1.0.0 - Portfolio Release

### Added

- Synthetic customer-success dataset and synthetic document corpus.
- Churn model training pipeline with saved scikit-learn artifact.
- Revenue-at-risk scoring engine.
- Hybrid retrieval interface with local TF-IDF fallback.
- Optional OpenAI-compatible LLM provider with mock fallback.
- Agent workflow with grounded answer, cited evidence, recommended actions, and email draft.
- Groundedness heuristic evaluation.
- RAG evaluation dataset and summary metrics.
- Human feedback logging.
- Account brief Markdown export.
- What-if simulator.
- FastAPI backend and Streamlit dashboard.
- Docker, Docker Compose, Makefile, GitHub Actions CI.
- Deployment documentation and Streamlit Community Cloud support files.

### Status

- Tests pass locally and in GitHub Actions.
- Public dashboard deployment is prepared but not completed in this environment because Streamlit Community Cloud requires a browser login.
